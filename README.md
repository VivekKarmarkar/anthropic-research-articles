# Anthropic Research Articles

A curated collection of Anthropic research articles — the original PDFs, AI-generated audio narrations, and NotebookLM podcast-style discussions — all in one place.

## Overview

This repo collects three Anthropic research articles exploring AI capabilities in physics, scientific computing, and emotion modeling. Each article is available as a source PDF, a Claude Code-generated audio narration (via TTS), and a NotebookLM-generated audio/video discussion.

## Project Structure

```
src_pdfs/                          # Original Anthropic research article PDFs
claude_code_generated_audio/       # TTS narrations generated by Claude Code + OpenAI TTS
notebookLM_outputs/
  notebookLM_audio/                # NotebookLM podcast-style audio discussions
  notebookLM_video/                # NotebookLM video summaries
others/                            # Miscellaneous files
```

## How the Audio Was Made

- **Claude Code audio**: PDFs were extracted to text, chunked, and synthesized to MP3 using OpenAI's `gpt-4o-mini-tts` model via a custom [audify skill](https://github.com/VivekKarmarkar) in Claude Code.
- **NotebookLM audio/video**: Generated using Google's NotebookLM, which creates podcast-style discussions from source documents.

## Note

Audio and video files (`.mp3`, `.m4a`, `.mp4`) are git-ignored due to their size. They are available via [Google Drive](https://drive.google.com) (shared links) or can be regenerated from the source PDFs.
